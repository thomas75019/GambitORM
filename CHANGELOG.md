## [0.1.0] - 2024-01-XX

### Added
- Initial release
- Database connection management (MySQL, PostgreSQL, SQLite)
- Query Builder with fluent API
- Model-based ORM with CRUD operations
- Database migrations system
- Schema builder for migrations
- Relationship support (hasOne, hasMany, belongsTo)
- Eager loading
- Join queries (INNER, LEFT, RIGHT, FULL)
- Transaction support
- Model validation with custom validators
- Lifecycle hooks (beforeSave, afterSave, beforeDelete, etc.)
- CLI tool for migration management
- Advanced query methods (whereIn, whereNull, whereBetween, subqueries, raw SQL)
- Aggregate functions (COUNT, SUM, AVG, MAX, MIN)
- TypeScript support
- Comprehensive test suite

### Features
- Support for MySQL, PostgreSQL, and SQLite
- Connection pooling
- Parameterized queries
- Type-safe models
- Migration rollback
- Validation error handling
- Hook priority system
- Subquery support
- Raw SQL execution
